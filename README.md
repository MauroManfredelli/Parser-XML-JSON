# Parser-XML-JSON
Progetto universitario: Parser per tradurre da xml a json

Dati:
• Una grammatica DTD che definisce una classe
di documenti scritti in XML
• Due documenti scritti nel linguaggio XML
conforme alla grammatica DTD fornita.

Utilizzare i tool JFlex e BYACC/J per:
Definire i file di input di JFlex e BYACC/J con cui
questi programmi realizzano un traduttore da
linguaggio XML (conforme al DTD fornito) a
linguaggio JSON.
